# BoldBox — Brand Reference

> *"Watch in seconds. Feel in moments. Return every day."*

---

## 1. Brand Foundation

### Brand Name
**BoldBox**

- **Bold** — กล้า ชัดเจน มีพลัง ไม่กลัวโดดเด่น
- **Box** — กล่อง / พื้นที่ / หน้าจอ (อ้างอิง set-top box และ black box ของโรงละคร)
- รวมกัน: *พื้นที่สำหรับเรื่องราวที่กล้าและทรงพลัง*

### Positioning Statement
> BoldBox is a micro drama streaming platform built for the attention economy — combining short-form video with storytelling to deliver instant emotional engagement and scalable monetization.

### Taglines

| บริบท | Tagline |
|---|---|
| Hero / Consumer | "Watch in seconds. Feel in moments. Return every day." |
| Business / Investor | "BoldBox turns short attention into storytelling, retention, and scalable monetization." |

### Vision
> BoldBox is building the next cross-border storytelling platform for Asia.

### Mission
สร้างแพลตฟอร์มที่ทำให้ local stories เดินทางข้ามประเทศได้ ผ่าน micro drama ที่เข้าถึงง่าย สนุก และเชื่อมผู้ชม แบรนด์ และวัฒนธรรมเข้าด้วยกัน

## 2. Brand Positioning

BoldBox อยู่ระหว่าง **TikTok** และ **Netflix** โดยตั้งใจ:

```
TikTok ←————————— BoldBox ————————→ Netflix
(เร็ว แต่ไม่มี story)            (มี story แต่ช้าเกินไป)
```

| มิติ | BoldBox | TikTok | Netflix |
|---|---|---|---|
| ความเร็ว | ✅ เร็ว Short-form | ✅ | ❌ ช้า |
| Storytelling | ✅ Scripted Drama | ❌ ตื้น | ✅ |
| Daily Habit | ✅ | ✅ | ❌ |
| Mobile-first | ✅ | ✅ | ❌ |

**Brand Archetype:** The Entertainer × The Explorer
— สร้างความสุข ความตื่นเต้น ขณะเดียวกันก็พาผู้ชมค้นพบเรื่องใหม่ข้ามวัฒนธรรม

## 3. Visual Identity

### Logo

โลโก้ประกอบด้วย 2 ส่วน:

**Icon Mark**
- รูปทรง Rounded Play Button แบบ organic (ไม่ใช่สามเหลี่ยมตรงๆ)
- ซ้อน 2 ชั้น: ชั้นหน้า (BoldBox Yellow) + ชั้นหลัง offset (Shadow Gold) → ให้ความรู้สึก depth / มีน้ำหนัก
- ตัวอักษร "B" สีขาว Bold อยู่ตรงกลาง
- อ้างอิง Play Button → ชัดเจนว่าเป็น entertainment platform

**Wordmark**
- "BoldBox" ฟอนต์ Rounded Bold Sans-serif (ใกล้เคียง Nunito ExtraBold / Poppins ExtraBold)
- สีเหลืองเดียวกับ icon mark
- วางใต้ icon บน dark background

### Color Palette

| ชื่อสี | Hex | การใช้งาน |
|---|---|---|
| **BoldBox Yellow** | `#F4C430` | Primary — logo, CTA, highlight, accent หลัก |
| **Shadow Gold** | `#A08C20` | Depth layer, shadow, hover state |
| **Pure Black** | `#000000` | Background หลัก, app base, brand page |
| **White** | `#FFFFFF` | Text บน dark, icon กลางโลโก้ |
| **Dark Surface** | `#1A1A1A` | Card bg, secondary surface |
| **Mid Gray** | `#666666` | Body text สำรอง, placeholder |
| **Success Green** | `#22C55E` | Confirmation, positive state |
| **Alert Red** | `#EF4444` | Error, notification badge |

### Typography (แนวทาง)

- **Primary Font:** Rounded Bold Sans-serif (Nunito / Poppins) — ใช้กับ headline, logo, CTA
- **Body Font:** Clean Sans-serif ที่รองรับภาษาไทย — Sarabun / IBM Plex Sans Thai
- **สไตล์:** Bold หัวข้อ, regular body — ไม่ใช้ฟอนต์บางหรือ light เป็นหลัก

### Visual Style Direction

- **Dark, cinematic, emotional** — รู้สึกเหมือนดูหนัง
- พื้นหลังมืด (Black / Dark surface) เป็นหลัก — streaming-native
- Typography ใหญ่ กล้า อ่านง่ายบนมือถือ
- Imagery: ใบหน้า อารมณ์ ดราม่า — ไม่ใช่ landscape หรือ stock photo ทั่วไป
- Motion: เร็ว dynamic — ไม่หนักหรือช้าเกินไป
- Yellow-on-Black เป็น signature visual ของแบรนด์

## 4. Brand Personality

### 5 คำที่อธิบาย BoldBox

| คุณสมบัติ | ความหมาย |
|---|---|
| **Bold** | กล้า ชัดเจน ไม่กลัวโดดเด่น |
| **Emotional** | สัมผัสอารมณ์ได้ เชื่อมกับผู้ชมจริงๆ |
| **Fast** | เร็ว ตรง ไม่เสียเวลาผู้ชม |
| **Warm** | เป็นกันเอง อบอุ่น ไม่เย็นชา |
| **Asian** | รากเหง้าชัด ภูมิใจในวัฒนธรรม ไม่พยายามเป็นตะวันตก |

### Tone of Voice

**พูดแบบเพื่อน ไม่ใช่บริษัทใหญ่**

- สั้น กระชับ ได้ใจความ — ทุกประโยคต้องสื่อสารได้ใน 1 อ่าน
- มี emotional pull — ให้รู้สึกว่า "นี่แหละคือเรื่องที่ฉันอยากดู"
- ใช้ภาษาไทยที่เป็นธรรมชาติ ไม่ formal ไม่ slang จนเกินไป
- ตื่นเต้นได้ แต่ไม่ดูเว่อร์

**ตัวอย่าง**

| ✅ ใช้แบบนี้ | ❌ ไม่ใช้แบบนี้ |
|---|---|
| "ดูจบใน 3 นาที อินได้ทั้งวัน" | "BoldBox มอบประสบการณ์การรับชมที่ครบครัน" |
| "เรื่องใหม่ รู้สึกใหม่ ทุกวัน" | "แพลตฟอร์มสตรีมมิ่งที่ตอบสนองทุกความต้องการ" |
| "เปิดแอป แล้วอินได้เลย" | "เข้าถึงคอนเทนต์คุณภาพสูงได้อย่างสะดวกสบาย" |

## 5. Target Audience

### Primary — Thai Mobile Entertainment User

- อายุ **18–35 ปี** (Gen Z + Millennials)
- Mobile-first, ดู TikTok / Reels / YouTube Shorts เป็นประจำ
- มีเวลาน้อย แต่อยากได้ความบันเทิงที่ "อิน" ได้เร็ว
- ชอบแนว Romance, Drama, BL/GL, Revenge, Fantasy, Workplace
- กลุ่มหลัก: **ผู้หญิงและ LGBTQ+** ที่มี emotional engagement สูงกับ scripted drama

### Secondary — Thai Casual Viewer

- พนักงานบริษัท / ฟรีแลนซ์ / คนเมือง
- ดูระหว่างเดินทาง พักกินข้าว ก่อนนอน
- ต้องการความบันเทิงที่ "จบได้" ในเวลาสั้น

### B2B — Brand & Advertiser

- แบรนด์ไทยที่ต้องการ story-driven advertising channel ใหม่
- ต้องการเข้าถึง Gen Z–Millennial แบบ mobile-first, highly engaged
- ต้องการพื้นที่ที่ไม่ใช่แค่ banner ad แต่เป็น emotional context

### Future — ASEAN

- อินโดนีเซีย, เวียดนาม, ฟิลิปปินส์, มาเลเซีย
- พฤติกรรมคล้ายกัน: mobile-first, drama lover, social video user

## 7. Brand Story (Short Version)

ในยุคที่ทุกคนมีเวลาน้อยลงแต่อยากอินกับเรื่องราวมากขึ้น — BoldBox เกิดขึ้นมาเพื่อเติมเต็มช่องว่างนั้น

ไม่เร็วแบบ TikTok จนไม่มีเรื่อง ไม่ช้าแบบ Netflix จนไม่มีเวลา
BoldBox คือพื้นที่ที่ดูแล้วได้อิน ดูแล้วอยากกลับมาอีก

เริ่มจากประเทศไทย — ขยายไปอาเซียน — เชื่อมโลกด้วยเรื่องราว
